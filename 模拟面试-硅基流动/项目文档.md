# AI模拟面试与能力提升软件 - 项目文档

## 一、项目概述

### 1.1 项目背景
随着互联网行业的快速发展，技术岗位的竞争日益激烈。面试作为求职的关键环节，对候选人的专业知识、逻辑思维、表达能力等综合素质提出了较高要求。然而，许多求职者在面试准备过程中面临以下痛点：
- 缺乏真实的面试场景练习机会
- 无法获得专业的面试反馈和指导
- 难以了解自身的知识盲点和薄弱环节
- 面试经验不足导致临场发挥失常

### 1.2 项目目标
本项目旨在开发一款基于大语言模型的AI模拟面试系统，为求职者提供：
- 真实、专业的模拟面试体验
- 个性化、详细的面试评估报告
- 针对性的能力提升建议
- 便捷的面试记录管理和回顾功能

### 1.3 核心功能
1. **智能模拟面试**：基于硅基流动大模型API，模拟真实面试官进行多轮问答
2. **多岗位支持**：支持Java、Python、前端等多种技术岗位的模拟面试
3. **详细评估反馈**：每道题提供包含优点、不足、改进建议的详细点评
4. **综合能力报告**：面试结束后生成包含能力维度分析、学习建议的总结报告
5. **面试记录管理**：登录用户可查看历史面试记录和评估报告
6. **PDF报告导出**：支持将面试报告导出为PDF格式，方便保存和分享

## 二、项目创意

### 2.1 创意描述
本项目创新性地将大语言模型技术应用于面试培训领域，打造了一款"AI面试官"产品。与传统面试准备方式相比，具有以下创新点：

#### 2.1.1 技术创新
- **大模型驱动**：基于硅基流动DeepSeek-V2大模型，具备强大的自然语言理解和生成能力
- **流式对话**：支持流式响应，提供接近真人的对话体验
- **智能评估**：AI能够根据回答内容自动判断合格与否，并给出专业评价

#### 2.1.2 产品创新
- **动态难度调整**：根据用户回答质量动态调整问题难度
- **个性化问题生成**：针对不同岗位和候选人表现生成定制化问题
- **场景化面试**：结合实际业务场景提问，减少纯理论"八股文"

#### 2.1.3 体验创新
- **进度可视化**：实时显示面试进度，让用户心中有数
- **详细反馈**：每道题都提供优点、不足、改进建议三部分点评
- **报告导出**：支持PDF导出，方便用户回顾和分享

### 2.2 市场需求分析

#### 2.2.1 目标用户
- **应届毕业生**：缺乏面试经验，需要系统练习
- **转行求职者**：需要了解新行业的面试要求
- **在职提升者**：希望检验自身技术水平，寻找提升方向

#### 2.2.2 市场规模
- 中国每年高校毕业生超过1000万
- IT行业求职者数量持续增长
- 在线职业教育市场规模超过5000亿元

### 2.3 社会价值
- **降低求职成本**：减少因面试失败带来的时间和经济成本
- **促进教育公平**：让优质面试培训资源惠及更多人群
- **提升人才质量**：帮助求职者明确提升方向，促进整体技术水平提升

### 3.3 核心模块设计

#### 3.3.1 用户认证模块
- **注册功能**：用户名+密码注册，自动登录
- **登录功能**：Session-based认证，支持Cookie持久化
- **状态管理**：Pinia存储登录状态，全局响应式
- **权限控制**：未登录用户无法开始面试

#### 3.3.2 面试会话模块
- **会话创建**：选择岗位后创建新面试会话
- **多轮对话**：支持多轮问答，保持上下文
- **进度管理**：实时跟踪面试进度（自我介绍不计入）
- **结束判定**：达到设定题数后自动结束

#### 3.3.3 评估反馈模块
- **单题评估**：每道题给出详细评价（优点、不足、建议）
- **合格判定**：根据回答质量判定是否合格
- **综合报告**：面试结束后生成综合能力评估报告
- **报告导出**：支持PDF格式导出

#### 3.3.4 历史记录模块
- **记录列表**：展示所有历史面试记录
- **详情查看**：可查看每次面试的详细问答和评估
- **状态筛选**：按完成状态筛选记录

```sql
    total_questions INTEGER DEFAULT 5,
    passing_threshold INTEGER DEFAULT 3,
    start_time DATETIME,
    end_time DATETIME,
    final_evaluation TEXT
);

CREATE TABLE qa_records (
    id INTEGER PRIMARY KEY,
    interview_id INTEGER FOREIGN KEY,
    question TEXT,
    answer TEXT,
    evaluation TEXT,
    is_passed BOOLEAN,
    created_at DATETIME
);
```

```python
    interview = Interview.objects.get(session_id=session_id)

    evaluation = evaluate_answer(interview.current_question, answer)
    # 生成下一个问题
    next_question = generate_question(interview.job_name)
    return evaluation, next_question
```

#### 4.1.3 PDF报告生成
```python
def generate_pdf_report(interview):
    doc = SimpleDocTemplate("面试报告.pdf")
    elements = []
    
    # 标题
    elements.append(Paragraph("面试评估报告", title_style))
    
    # 基本信息
    elements.append(Paragraph(f"岗位：{interview.job_name}", normal_style))
    elements.append(Paragraph(f"时间：{interview.start_time}", normal_style))
    
    # 问答详情
    for qa in interview.qa_records:
        elements.append(Paragraph(f"问题{qa.index}：{qa.question}", question_style))
        elements.append(Paragraph(f"回答：{qa.answer}", answer_style))
        elements.append(Paragraph(f"评价：{qa.evaluation}", evaluation_style))
    
    # 综合总结
    elements.append(Paragraph("面试总结", subtitle_style))
    elements.append(Paragraph(interview.final_evaluation, normal_style))
    
    doc.build(elements)
```
